Full Stack Lab is a laboratory where I try some technologies from the back-end to the front-end of the full stack.

## Current technologies
  * Java
  * Gradle
  * JUnit
  * JaCoCo
  * Spring Boot
  * Docker
  * JFrog Bintray
  * Travis CI
  * Coveralls

## Future technologies
  * AngularJS
  * Gulp
  * Bower
  * Twitter Bootstrap
  * MongoDB
  * Logback
  * Apache Corbova
